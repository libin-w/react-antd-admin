### useEventAwayTargets

设置页面标题。

#### Attributes

| 参数    | 说明     | 类型      | 是否必填 | 默认值 |
| ------- | -------- | --------- | -------- | ------ |
| value   | 页面标题 | `string`  | 是       | -      |
| options | 扩展配置 | `Options` | 否       | -      |

##### Options

| 参数             | 说明                       | 类型      | 是否必填 | 默认值 |
| ---------------- | -------------------------- | --------- | -------- | ------ |
| restoreOnUnmount | 组件卸载后恢复页面标题     | `boolean` | 否       | false  |
| validString      | 传入值是否必须为有效字符串 | `boolean` | 否       | false  |
